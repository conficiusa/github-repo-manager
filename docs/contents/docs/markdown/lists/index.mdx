# Advanced Usage Guide

## Best Practices

### Repository Management

1. **Use Dry Run First**
   - Always use `--dry-run` before actual operations
   - Review the list of repositories to be managed
   - Double-check pattern matching results

2. **Pattern Matching**
   - Use specific patterns to target repositories
   - Test patterns with small subsets first
   - Examples of effective patterns:
     ```bash
     # Match test repositories
     github-repo-manager --pattern "test-.*"
     
     # Match dated repositories
     github-repo-manager --pattern ".*-2023-.*"
     ```

3. **Batch Processing**
   - Use delay parameter for large operations
   - Consider GitHub API rate limits
   - Example with delay:
     ```bash
     github-repo-manager --delay 2.0
     ```

## Error Handling

### Common Issues and Solutions

1. **Authentication Errors**
   ```python
   # Proper token handling
   import os
   from github_repo_manager import manage_repositories

   token = os.environ.get('GITHUB_TOKEN')
   manage_repositories(token=token)
   ```

2. **Rate Limiting**
   - Use appropriate delays
   - Handle rate limit exceptions
   - Monitor API quota usage

3. **Pattern Matching Issues**
   - Use proper regex patterns
   - Test patterns before operations
   - Common pattern examples:
     ```python
     # Match exact prefix
     pattern = "^test-"
     
     # Match specific date format
     pattern = ".*-2023(0[1-9]|1[0-2])-.*"
     ```

## Advanced Integration

### Using as a Library

```python
from github_repo_manager import GitHubRepoManager

# Initialize with custom settings
manager = GitHubRepoManager(
    token="your-token",
    username="your-username"
)

# Get filtered repositories
repos = manager.get_repositories(
    filter_pattern="test-",
    visibility="private"
)

# Custom processing
for repo in repos:
    # Add your custom logic here
    if some_condition(repo):
        manager.manage_repository(repo['name'])
```

### Environment Variables

Set up environment variables for secure token storage:
```bash
export GITHUB_TOKEN="your-token"
export GITHUB_USERNAME="your-username"
```

### Automated Scripts

Example of automated cleanup script:
```python
#!/usr/bin/env python3
from github_repo_manager import manage_repositories
import logging
import sys

logging.basicConfig(level=logging.INFO)

def cleanup_old_repos():
    try:
        manage_repositories(
            pattern="test-.*",
            dry_run=True,  # Remove for actual operations
            no_confirm=True
        )
    except Exception as e:
        logging.error(f"Cleanup failed: {e}")
        sys.exit(1)

if __name__ == "__main__":
    cleanup_old_repos()
```
